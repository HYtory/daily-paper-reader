## 摘要
我们研究了多智能体深度强化学习，并对多智能体深度确定性策略梯度（MADDPG）算法提出了两项改进。首先，我们引入了一种新颖的动作推理机制，使每个智能体能够预测其他智能体的意图动作，从而提高自身策略的准确性和稳定性。其次，我们在回放缓冲区中应用了基于几何分布的重要性采样策略，优先考虑更新近和更有信息量的经验，这有助于缓解多智能体环境中固有的非平稳性。我们在PettingZoo库提供的离散动作捕食者-猎物任务上评估了这两项改进，PettingZoo是一个灵活的Python接口，用于通用多智能体强化学习基准测试。我们的结果表明，动作推理能有效提高学习稳定性和智能体间合作，而使用几何分布的重要性采样在探索效率上比标准MADDPG有显著提升。代码可在https://github.com/shaashwathsivakumar/MARL_Proj获取。

## Abstract
We investigate multi-agent deep reinforcement learning and propose two enhancements to the Multi-Agent Deep Deterministic Policy Gradient (MADDPG) algorithm. First, we introduce a novel Action Inference mechanism that enables each agent to predict other agents' intended actions, thereby improving the accuracy and stability of its own policy. Second, we apply an importance sampling strategy, using geometric distribution, in the replay buffer to prioritize more recent and informative experiences, which helps mitigate the non-stationarity inherent in multi-agent environments. We evaluate both modifications on the discrete-action Predator-Prey task provided by the PettingZoo library, a flexible Python interface for general multi-agent reinforcement learning benchmarks. Our results indicate that Action Inference is effective in improving learning stability and inter-agent cooperation and that importance sampling using geometric distribution can lead to significant improvements in exploration efficiency over standard MADDPG. Code available at https://github.com/shaashwathsivakumar/MARL_Proj

## 2. 论文提出的方法论
### 核心思想
- **动作推理（Action Inference）**：每个智能体利用一个预训练的网络（AI Net）预测所有智能体的历史动作，并将预测结果作为额外输入馈送给自己的Actor网络，从而提供关于环境转移的上下文信息，隐式实现智能体间通信。
- **几何分布重要性采样（Geometric Importance Sampling）**：替换标准MADDPG中的均匀回放采样，用几何分布对经验索引进行采样，使更近期的经验被优先抽取，以更快响应环境动态变化。

### 关键技术细节
- **动作推理网络架构**：针对可分离观测（即每个智能体观测可分解为部分），设计**方向性社会感知模块**（用于预测其他智能体动作）和**方向性自我感知模块**（用于预测自身历史动作）。通过时间堆叠观测并计算差分，输入模块，输出对每个智能体one-hot动作的预测。网络预训练在随机或大量回放数据上进行，训练完成后固定参数。
- **集成方式**：在MADDPG训练循环中，每个时间步，Agent i 使用已训练的AI Net根据当前观测和上一时间步观测生成预测动作向量 \(\hat{a}_i\)，然后将 \(\hat{a}_i\) 作为Actor网络的附加输入。Critic更新和Actor策略梯度计算均使用原始真实动作，不增加优化复杂度。
- **几何分布采样**：几何分布概率质量函数 \(P(X=k) = (1-p)^{k-1}p\)，\(p\) 控制近期偏置强度。将采样索引作为回放缓冲区的反向索引，使越新的经验被抽中概率越高。实验中 \(p\) 取固定值 \(10^{-5}\)，缓冲区大小为 \(7.5\times10^5\)。

### 算法流程（文字说明）
- **标准MADDPG**（Algorithm 1）：每个episode，对每个时间步，每个Agent根据自身观测选择动作（加噪声），执行后存储经验元组 \((x, a, r, x')\)，随后采样小批量更新Critic（最小化Bellman误差）和Actor（通过确定性策略梯度），并软更新目标网络。
- **预训练动作推理**（Algorithm 2）：在随机策略或已有轨迹上，收集观测对 \((o, o^-)\)，对每个智能体i，分别用自我模块（预测自身动作）和社会模块（预测其他智能体动作）进行监督学习，损失为one-hot动作的MSE。
- **MADDPG+动作推理**（Algorithm 3）：第一个时间步仅使用零向量作为\(\hat{a}_i\)；从第二个时间步起，先调用AI Net得到预测，再让Actor基于\((o_i, \hat{a}_i)\)选择动作。存储经验时额外保存\(o\)和\(o^-\)。Critic更新时使用上一时间步的观测预测下一时间步的\(\hat{a}'\)，用于计算目标Q值。Actor梯度计算也依赖当前\(\hat{a}_i\)。
