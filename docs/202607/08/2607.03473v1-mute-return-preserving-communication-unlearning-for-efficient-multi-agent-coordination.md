## 摘要
智能体间通信对于协调部分可观测环境下的多智能体强化学习（MARL）智能体在合作博弈中有效执行至关重要；然而，现实世界的带宽限制要求稀疏交互。先前的方法主要通过优化信息论替代指标来处理这种权衡。我们认为这些统计代理从根本上与真实目标不一致：一条消息可能信息量很大，但与任务的联合回报无关。在这项工作中，我们提出面向目标效率的消息遗忘（MUTE），一个将通信减少视为价值引导的机器遗忘问题的框架。MUTE 使用基于注意力的估计器严格量化反事实消息价值，并系统地从无任何通信约束训练的策略中遗忘低价值消息的传输。这是通过一个双目标机制实现的，该机制强制通信稀疏性，同时保持原始联合策略的回报。我们推导了由这种稀疏化引起的性能差距的理论上界，保证了可控的回报退化。我们还在各种复杂的多智能体环境中对 MUTE 进行了实证评估，实现了 80% 到 90% 的带宽降低，同时保持了与最先进基线相当的性能。

## Abstract
Inter-agent communication is critical for coordinating Multi-Agent Reinforcement Learning (MARL) agents under partial observability to perform effectively in cooperative games; however, real-world bandwidth constraints demand sparse interactions. Prior approaches primarily address this trade-off by optimizing information-theoretic surrogates. We argue that these statistical proxies are fundamentally misaligned with the true objective: a message can be highly informative yet irrelevant to the joint return of the task. In this work, we propose Message Unlearning for Targeted Efficiency (MUTE), a framework that views communication reduction as a value-guided machine unlearning problem. MUTE rigorously quantifies the Counterfactual Message Value using an attention-based estimator, and systematically unlearns the transmission of low-value messages from a policy trained without any communication constraints. This is achieved through a dual-objective mechanism that enforces communication sparsity while preserving the return of the original joint policy. We derive a theoretical upper bound on the performance gap induced by this sparsification, guaranteeing controlled return degradation. We also empirically evaluate MUTE on various complex multi-agent environments, achieving 80% to 90% bandwidth reduction while maintaining performance comparable to state-of-the-art baselines.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：在多智能体强化学习（MARL）中，智能体间通信对于克服部分可观测性、实现有效协调至关重要，但现实世界中的带宽限制要求通信必须稀疏。现有方法主要依赖信息论代理目标（如最小化互信息）来压缩通信，但这些统计代理与真正目标——最大化联合回报——从根本上存在偏差。一条消息可能信息量很高（例如能强烈改变接收者行为的消息），但对联合回报可能是负面的，从而导致协调退化。

- **整体含义**：作者论证，通信缩减不应被看作一个平衡回报与通信代价的多目标优化问题，而应被视为一个**回报保持（return-preserving）的冗余消除问题**。即先训练一个全通信专家策略，然后识别并遗忘那些对回报无贡献的低价值消息，同时严格保证性能损失可控。

## 2. 提出的方法论

### 核心思想
将通信缩减建模为**价值引导的机器遗忘（value-guided machine unlearning）** 问题：从一个无通信约束的收敛专家策略出发，利用反事实消息价值（Counterfactual Message Value, CMV）量化每条消息对联合回报的边际贡献，然后通过**双目标损失**（稀疏损失 + 行为锚定损失）系统性地遗忘低价值消息，同时将策略行为锚定回原始专家策略，确保回报不退化。

### 关键技术细节
- **反事实消息价值（CMV）**：定义为，在给定状态-历史下，发送消息时产生的联合动作价值与屏蔽该消息后重新采样所有接收者动作产生的联合动作价值之差。若CMV≈0，则该消息是冗余的。
- **消息价值估计器（Message Value Estimator, MVE）**：采用自注意力架构，一次性处理所有智能体的消息集合，输出每条消息的CMV估计值。注意力机制天然满足排列不变性和上下文依赖性。
- **通信遗忘双目标损失**：
  - **稀疏损失** `L_sparse`：对CMV低于动态阈值 `λ·μ_MVE` 的消息施加L1范数惩罚，连续梯度压力迫使低价值消息幅值衰减。
  - **行为锚定损失** `L_anchor`：最小化当前策略各个智能体的个体Q值与冻结的专家个体Q值之间的均方误差，防止策略在通信通道被修剪时崩溃。
  - 联合优化：`L = L_sparse + β·L_anchor`，在更新策略参数和消息生成器的同时保持稀疏与稳定。

### 性能上界定理
在假设MVE误差有界（≤δ）、动作值估计误差有界（≤ϵ）的前提下，推导出性能差距上界：
`J(π*) - J(π) ≤ (1/(1-γ)) * E[ 2ϵ + |M_red|(λ+δ) + L·Σ∥mi∥² ]`，保证了由遗忘引起的回报退化可控。

### 算法流程（三阶段）
1. **预训练阶段**（2M步）：在无通信限制下训练专家策略（基于MASIA框架），获得收敛的Q*tot和Q*i。
2. **MVE训练阶段**（0.5M步）：利用经验回放中的数据，通过采样反事实基线训练消息价值估计器。
3. **通信遗忘阶段**（1.5M步）：冻结专家策略，联合优化稀疏损失和行为锚定损失，逐渐遗忘低CMV消息。

## 6. 主要结论与发现

- MUTE在几乎所有10个环境上实现了**80%~90%的带宽缩减**（最终通信率低至1%~10%），同时保持与全通信基线MASIA相当甚至更优的胜率。

- 消融实验证明：
  - 缺少锚定损失会导致策略立即崩溃。

- 理论性能上界在实践中得到验证（Q误差和MVE损失均保持低水平）。

## 7. 优点

- **问题框架新颖**：将通信缩减重新定义为“回报保持的机器遗忘”问题，避免了传统方法中回报与通信代价的模糊权衡。
- **方法可解释性好**：CMV直接衡量消息对联合回报的边际贡献，而非信息论代理。
- **计算高效**：注意力型MVE可一次性为所有智能体估计消息值，避免了Shapley值所需的指数级枚举。
- **理论保证**：提供了性能退化的上界，为安全遗忘提供理论支撑。
- **实验扎实**：在多种复杂环境（包括高维SMAC、随机化SMACv2）上验证，消融全面，超参数敏感度分析完整。

## 8. 不足与局限

- **CMV近似局限**：CMV仅评估单个消息被屏蔽时的影响，并未显式枚举所有消息子集，可能无法完全捕捉高阶交互效应（例如多条冗余消息被同时屏蔽时的影响）。作者承认此局限，并指出行为锚定损失可在一定程度上补偿，但未提供更严格的子集感知估计方法。
- **训练阶段依赖**：需要三阶段训练（预训练→MVE训练→遗忘），增加了实现复杂度。虽然总步数与其他方法一致，但实际训练流程更繁琐。
- **对某些环境性能不如无通信方法**：在SMACv2 Terran 10v10上，MUTE胜率不如QMIX和MAPPO（由于环境随机化削弱了通信优势）。作者已分析原因，但表明MUTE在该类场景中优势不明显。
- **资源报告不完整**：未披露GPU数量、总计算开销等，难以复现或横向比较算力需求。
- **遗忘阈值λ依赖超参数**：虽然通过指数移动平均实现了自适应阈值，但仍需手动调节λ（默认1.0），且在不同环境上表现敏感（见附录H）。
